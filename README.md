# ResearchGPT — Milad Saeedi

A retrieval-augmented portfolio assistant for exploring Milad Saeedi's
research, thesis, publications, machine-learning experience, and selected
GitHub projects.

## Required Space secret

Add the following private secret in **Settings → Variables and secrets**:

```text
GEMINI_API_KEY
```

Optional variable:

```text
GEMINI_MODEL=gemini-3.5-flash
```
